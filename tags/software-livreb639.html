Para deixar o editor vim colorido é muito fácil.
Basta mover um arquivo texto para o home do usuário e pronto hehee.
Com o usuário do seu sistema que vai ficar com o vim colorido, entre nesse diretório:
 $cd /usr/share/vim/vim63
Liste esse diretório e copie esse arquivo para o home do usuário que você queres que fique com o vim colorido:
 #cp vimrc_example.vim ~/.vimrcleia mais
